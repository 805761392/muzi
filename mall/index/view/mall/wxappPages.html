页面参数使用示例：订单列表，需要查询状态是已完成的订单，则拼接参数后的路径为pages/order/order?status=3
小程序页面
  • 商城首页
    路径： pages/index/index
  • 分类
    路径： pages/cat/cat
  • 购物车
    路径： pages/cart/cart
  • 用户中心
    路径： pages/user/user
  • 商品列表
    路径： pages/list/list
    参数： cat_id 分类id
  • 搜索页
    路径： pages/search/search
  • 商品详情
    路径： pages/goods/goods
    参数： id 商品id
  • 订单列表
    路径： pages/order/order
    参数： status 订单状态[所有订单=-1,待付款=0,待发货=1,待收货=2,已完成=3,售后=4]
  • 我的收藏
    路径： pages/favorite/favorite
  • 分销中心
    路径： pages/share/index
  • 关于我们
    路径： pages/article-detail/article-detail
    参数： id=about_us
  • 服务中心
    路径： pages/article-list/article-list
    参数： id=2
  • 我的优惠券
    路径： pages/coupon/coupon
  • 门店列表
    路径： pages/shop/shop
  • 视频列表
    路径： pages/video/video-list
  • 领券中心
    路径： pages/coupon-list/coupon-list
  • 专题列表
    路径： pages/topic-list/topic-list
  • 专题详情
    路径： pages/topic/topic
    参数： id 专题id
  • 秒杀
    路径： pages/miaosha/miaosha
  • 小程序内嵌网页
    路径： pages/web/web
    参数： url 网址(网址必须UrlEncode，域名必须在小程序官方后台设置业务域名)
  • 门店详情
    路径： pages/shop-detail/shop-detail
    参数： shop_id 门店id
  • 我的卡券
    路径： pages/card/card
    参数： status 卡券状态(选填)[已失效=2]
  • 拼团首页
    路径： pages/pt/index/index
  • 拼团商品详情
    路径： pages/pt/details/details
    参数： gid 拼团商品id
  • 拼团订单列表
    路径： pages/pt/order/order
    参数： status 订单状态(选填)[全部=-1,待付款=0,拼团中=1,拼团成功=2,拼团失败=3,售后=4]
  • 预约首页
    路径： pages/book/index/index
  • 预约详情
    路径： pages/book/details/details
    参数： id 预约id
  • 我的预约列表
    路径： pages/book/order/order
    参数： status 状态(选填)[待支付=0,待使用=1,已使用=2,退款=3]
{/block}
